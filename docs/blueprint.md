# **App Name**: Sporting Club Manager

## Core Features:

- User Authentication: Secure login system with role-based access control for administrators and promoters.
- Dashboard: Display of key metrics like active students, attendance, recent activity, and pending requests.
- Confirmation Requests Management: Handle confirmation requests with detailed views, evidence upload, and status tracking.
- Packages Management: Create and manage training packages.
- Seasons Management: Manage school seasons, create new seasons and modify or disable seasons.
- Categories Management: Categorize student depending on their level/age and other parameters.
- Inventory Reports: Create inventory and inventory reports

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and stability, reflecting the sporting environment.
- Background color: Light gray (#EEEEEE) to provide a clean and modern feel.
- Accent color: Vibrant orange (#FF5722) to highlight key actions and calls to action, energizing the user experience.
- Body and headline font: 'PT Sans', a humanist sans-serif, for both headings and body text providing a balance of modern look with approachability. It's versatility allows a wide range of usage and readability.
- Use clear, modern icons that align with the modules and actions available in the system.
- Module-based design with top navigation, clear content sections, and prioritized information display.
- Subtle transitions and animations to enhance user interaction and provide feedback on actions.
